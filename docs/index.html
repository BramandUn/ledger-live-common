▸ fetchHistodayRates(currencies, against, getLatestDayFetched)
src/api/countervalue.js

Fetch countervalue rates history for crypto currencies (or a currency) per day granularity.

fetchHistodayRates(currencies: Array<Currency>, against: Currency, getLatestDayFetched: function (Currency): string?): Promise<CounterValuesPairing<Histoday>>
Parameters
currencies (Array<Currency>)
against (Currency)
getLatestDayFetched (function (Currency): string?) will returns for a currency the latest day that is already loaded and don't need to be refreshed. It returns this in YYYY-MM-DD format. If data was never loaded OR all data needs to be refetched, you can return a falsy value. NOTE: This allow implementation to do two things:
  • don't fetch anything if current day is the latest
  • only fetch part of the data that are missing (patch)
Returns
Promise<CounterValuesPairing<Histoday>>: a subset of pairs with only the new data (regarding getLatestDayFetched)
